# Popular WinGet packages for bulk icon-extraction CI runs.
# One PackageId per line. '#' starts a comment. Blank lines ignored.
#
# Curation rules:
#   - Must have a real ARP entry (no MSIX/Store, no portable-only).
#   - Silent install must work without a license key or interactive EULA prompt
#     beyond what --accept-package-agreements covers.
#   - No kernel drivers, no reboot-required installers, nothing >2 GB.
#   - Mix across categories so the artifact exercises a variety of installer
#     types (Inno, NSIS, MSI, Burn, Squirrel, custom).
